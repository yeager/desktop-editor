# Desktop Editor

Visual editor for .desktop entry files.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install desktop-editor
```

### Fedora/RPM
```bash
sudo dnf install desktop-editor
```
